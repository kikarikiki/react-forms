# React Forms Project

<ul>

  <li>Input Tags, Textarea Tags, Select Tags, Checkboxes, Radio Buttons</li>
  <li>Handling Multiple Inputs</li>
</ul>
styling it with the help of 
<ul>
  <li>Bootstrap 5</li>
  <li>Sass/Scss</li>
</ul>
